# AI Resume Screening Agent

## Overview
An AI-powered ATS system built using n8n, OpenAI, Google Sheets and Google Drive.

## Features

- Resume Upload
- PDF Parsing
- Candidate Information Extraction
- ATS Score Generation
- Skill Matching
- Hiring Recommendation
- Google Sheets Integration

## Tech Stack

- n8n
- OpenAI
- Google Sheets
- Google Drive
- JSON
- LLM

## Workflow

1. Candidate uploads resume
2. PDF extracted
3. Candidate data extracted
4. Job requirements fetched
5. AI evaluates candidate
6. ATS score generated
7. Results saved in Google Sheets# AI-Resume-Screening-Agent
